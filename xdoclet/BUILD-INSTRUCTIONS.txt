To build hacked-xdoclet, you need:

- Ant (http://ant.apache.org/)
- Create the file genesis/xdoclet/build.properties

In genesis/xdoclet/build.properties you must define the XDoclet Home (XDoclet 1.2)

   xdoclet.home=<XDOCLET_HOME>

e.g.
   xdoclet.home=/usr/xdoclet
   
You can download XDoclet 1.2 distribution at http://xdoclet.sourceforge.net/


Run ant to build the hacked xdoclet distribution.

e.g.:
   ant
     or
   ant xdoclet
