genesis' Downloads

There are several distribution bundles available for downloading. They are:

  • genesis-docs-{version}.tar.gz and genesis-docs-{version}.zip: genesis documentation.
  • genesis-bin-{version}.tar.gz and genesis-bin-{version}.zip: genesis binary distribution. This archive contains genesis jars and an empty project which can be used to start a new genesis application. This archive does not contain any jars genesis depends on, such as Hibernate.
  • genesis-full-bin-{version}.tar.gz and genesis-full-bin-{version}.zip: genesis full binary distribution. This archive contains the following files: all files contained in genesis-bin, including the empty project, and all the necessary jar files for genesis dependencies, such as AspectWerkz, Hibernate, Jakarta Commons and Thinlet.
  • genesis-full-bin-plus-docs-{version}.tar.gz and genesis-full-bin-plus-docs-{version}.zip: genesis full binary distribution and documentation. This archive contais all jars contained in genesis-full-bin, the empty project structure and complete genesis documentation, as well as some additional genesis samples, such as a simple calculator (SimpleCalculator) and a user administration application (Useradmin).
  • genesis-useradmin-sample-{version}.tar.gz and genesis-useradmin-sample-{version}.zip: contain both source code as well as binary files for the UserAdmin sample.
  • genesis-src-{version}.tar.gz and genesis-src-{version}.zip: genesis source code.
  • genesis-src-plus-docs-{version}.tar.gz e genesis-src-plus-docs-{version}.zip: genesis source code and genesis complete documentation.
  • modified-xdoclet-1.2.zip: modified xdoclet 1.2 distribution files with additional tasks.

All distribution files for this release can be found here.
