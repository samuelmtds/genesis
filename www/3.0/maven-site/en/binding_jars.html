Defining what jars are needed for your application

It is possible to use only the genesis binding features without using the empty project that comes with the framework distribution. However, what jars are needed vary depending on features and minimum JRE release to be supported. The table below describes requirements for each scenario:

commons-beanutils-<version>.jar

Annotation support genesis-annotation-jdk14-<version>.jar genesis-annotation-jdk5-<version>.jar
Swing binding genesis-client-swing-<version>.jar
SWT binding genesis-client-swt-<version>.jar
Thinlet binding genesis-client-thinlet-<version>.jar
