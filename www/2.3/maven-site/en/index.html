Overview

genesis is an open-source framework that aims to bring simplicity and productivity to enterprise application development, ensuring scalability, robustness and testability of your software. The main goal is to simplify the development of business components and the construction of complex graphical interfaces having the minimum effort for the developers. To accomplish its mission, genesis combines several open-source frameworks in a completely transparent way for the developer, through the use of AOP (Aspect Oriented Programming). Our main challenge is to allow people having little knowledge of the Java platform to be capable of develop robust applications fast, no new technologies and complex concepts learning required. We invite you to enjoy the benefits.

Where to start ?

Developers

Download genesis and read the User manual. Run the samples and start developing using the ready-to-use project structure. You should be creating your own applications in a while. If you have any questions, read our FAQ and join our mailing list. To subscribe to our mailing list, just send an e-mail to users-subscribe@genesis.dev.java.net. Then, just reply confirmation e-mail of the server. To send any message to our list, use the address users@genesis.dev.java.net.

Framework Developers

To understand genesis architecture better, please, refer to advanced documentation. Find out details about technologies of this framework (AOP, IOC, etc).
