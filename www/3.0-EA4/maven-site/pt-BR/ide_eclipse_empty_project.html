Passo a Passo para Importar o empty project no Eclipse

1. Clique com o botão direito no Package Explorer e escolha New Project

2. Escolha Java Project e clique em Next

3. Dê um nome para seu projeto e escolha a opção Create project from existing source, e escolha o diretório onde o empty-project se encontra. Depois clique em Next

4. Na tab Source escolha o diretório modules/shared/src e clique para adicionar como diretório de fontes (ícone com sinal de +). O diretório modules/client/src é automaticamente reconhecido pelo Eclipse como diretório de fontes. Coloque o Default output folder para um algum diretório qualquer como por exemplo <SEU_PROJETO>/bin.

5. Na tab Libraries clique em Add External Jars, e escolha os jars do genesis e os jars que estão na pasta lib do genesis.

6. Não é necessário adicionar todos os jars que você encontrar. Você pode escolher inicialmente os jars como mostrado na imagem. Se precisar de outro, você pode adicioná-lo mais tarde. Se for usar Java 5, adicione também a biblioteca genesis-annotation-jdk5-<versão>.jar

7. Agora você precisa fazer um cópia do build.properties.sample, clicando com o botão direito nele, e escolhendo Copy. Depois clique com o botão direito e escolha Paste. Nesse momento você vai dar o novo nome do arquivo, onde você deve colocar build.properties



8. Clique em build.properties com o botão direito, e escolha Open With - Properties File Editor.

9. Você abrirá um arquivo de propriedades, onde você deverá configurar as propriedades conforme o seu ambiente:

  • jboss.home - diretório do jboss

10. Depois de configurar as propriedades, você já pode compilar o seu projeto. Vá no menu Window - Show View - Ant

11. Arraste o arquivo build.xml para a view do ant. Dê 2 cliques na tarefa all que seu projeto será compilado.
