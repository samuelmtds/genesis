Introduction

This documentation explains genesis' main concepts and features so those can be applied to development. It is not expected that users will understand details about genesis' internal implementation/architecture, nor be capable of extending genesis features.

Besides specific documentation for each genesis module, there's an annotations reference guide to help developers with their syntax. Reading the other docs is still needed to understand how features actually work.

It's worth mentioning that although samples are configured to be run "as is" in JBoss application server and that docs sometimes refer to this product, all framework features are product-independent and J2EE- compatible.
