Posso utilizar o genesis para criar uma aplicaçao Web?

A atual versão do genesis já pode ser utilizada com qualquer framework Web, tais como: Struts ou WebWork.

Para criar uma aplicação Web basta construir um projeto que utilize os componentes de negócio do genesis.

Por que estou obtendo um NullPointerException numa linha cujo código possui um getSession()?

Há várias situações em que ocorrem este problema. Basicamente são elas:

  • Houve um encadeamento de chamadas entre os objetos de negócio. Neste caso, a session do Hibernate não foi injetada nas chamadas subsequentes. Para resolver isso, basta utilizar o método setSession() do objeto de negócio encadeado passando como parâmetroa própria session do objeto chamador.
  • Um objeto de negócio não se encontra na árvore de compilação client. Deste modo, o weaving não é realizado e a sessão do Hibernate não é injetada no objeto de negócio.
