Downloads do genesis

Para fazer o download do genesis, há várias opções disponíveis. São elas:

  • genesis-docs-{versão}.tar.gz e genesis-docs-{versão}.zip: contém apenas a documentação do genesis.
  • genesis-bin-{versão}.tar.gz e genesis-bin-{versão}.zip: contém a distribuição binária do genesis. Este empacotamento contém os seguintes grupos de arquivos: os jars do genesis propriamente ditos e um projeto vazio para iniciar o desenvolvimento de uma aplicação com o genesis. Ou seja, este empacotamento não possui nenhum arquivo jar de dependência a frameworks externos, tais como Hibernate.
  • genesis-full-bin-{versão}.tar.gz e genesis-full-bin-{versão}.zip: contém a distribuição binária completa do genesis. Este empacotamento contém os seguintes grupos de arquivos: todos os arquivos do genesis-bin, incluindo o projeto vazio, e as bibliotecas referentes às dependências do genesis, como AspectWerkz, Hibernate, Jakarta Commons e Thinlet.
  • genesis-full-bin-plus-docs-{versão}.tar.gz e genesis-full-bin-plus-docs-{versão}.zip: contém a distribuição binária completa do genesis juntamente com a documentação. Em termos de pacotes, são fornecidos os seguintes grupos de arquivos: todos os jars do genesis-full-bin, o projeto vazio para iniciar o desenvolvimento de uma aplicação com o genesis, o conteúdo de genesis-docs e outros exemplos de utilização do genesis, como uma simples calculadora (SimpleCalculator) e um aplicativo de administração de usuários (UserAdmin)
  • genesis-useradmin-sample-{versão}.tar.gz e genesis-useradmin-sample-{versão}.zip: contém tanto o código fonte como os binários de distribuição do exemplo UserAdmin.
  • genesis-src-{versão}.tar.gz e genesis-src-{versão}.zip: contém o código fonte do genesis.
  • genesis-src-plus-docs-{versão}.tar.gz e genesis-src-plus-docs-{versão}.zip: contém o código fonte do genesis juntamente com a documentação.
  • modified-xdoclet-1.2.zip: contém o framework xdoclet modificado para executar tarefas adicionais.
