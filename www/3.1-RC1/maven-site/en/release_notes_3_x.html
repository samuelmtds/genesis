3.0-RC1

Downloads

All distribution files for this release can be found here.

Features and changes

Some of the most noteworthy new features and changes are:

  • Binding strategies are now supported.
  • Binding for checkboxes and radiobuttons has been reviewed and documented.

The following bugs have been fixed on this release:

  • Validation is broken for Swing and SWT bindings (Issue # 399).
  • genesis.equals(any,any) fails with Java SE 6 (Issue # 419).
  • Binding of checkboxes/radiobuttons/toggle buttons is broken for Thinlet and SWT (Issue # 425 and issue # 433).
  • A StackOverflow is thrown if indexField is used with a @DataProvider bound to a JTable (Issue # 426).

The full change log between 3.0-EA5 and 3.0-RC1 can be found here.

The full documentation for this release can be found here.

3.0-EA5

Downloads

All distribution files for this release can be found here.

Features and changes

Some of the most noteworthy new features and changes are:

  • Error messages for malformed/misplaced annotations, non-existing properties and missing files have been improved.
  • The UserAdmin sample has been improved to handle better cases such as deleting a role associated with users and when SWT is not properly configured.

The following bugs have been fixed on this release:

  • all.with.webstart fails if the SWT dir does not exist (Issue # 376).
  • webstart-jnlp-local fails if jdbc.driver is overridden (Issue # 387).
  • Random NullPointerException thrown while reading annotations (Issue # 389).
  • JMenu/JMenuItem instances cannot be found by LookupStrategy (Issue # 394).
  • Validation fails for state with Swing (Issue # 401).

The full change log between 3.0-EA4 and 3.0-EA5 can be found here.

The full documentation for this release can be found here.

Features and changes

Some of the most noteworthy new features and changes are:

  • It's possible to use genesis-client in regular Java projects without using AOP.

The following bugs have been fixed on this release:
