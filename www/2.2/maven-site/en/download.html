genesis' Downloads

There are several distribution bundles available for downloading. They are:

  • genesis-dist-{version}.tar.gz and genesis-dist-{version}.zip: genesis distribution jars and the annotation.properties file.
  • genesis-dependencies-{version}.tar.gz and genesis-dependencies-{version}.zip: all the jar files for genesis dependencies, such as AspectWerkz, Hibernate, Jakarta Commons and Thinlet.
  • genesis-empty-project-{version}.tar.gz and genesis-empty-project-{version}.zip: comes with an empty project structure so it can be used to start a new genesis-based project.
  • genesis-useradmin-sample-{version}.tar.gz and genesis-useradmin-sample-{version}.zip: comes with the useradmin genesis sample application.
  • genesis-src-{version}.tar.gz and genesis-src-{version}.zip: comes with genesis source code only (no additional jar files).
  • genesis-docs-{version}.tar.gz and genesis-docs-{version}.zip: only genesis documentation.
  • genesis-xdoclet-extensions.tar.gz and genesis-xdoclet-extensions.zip: modified xdoclet distribution files with additional tasks.
  • genesis-all-{version}.tar.gz and genesis-all-{version}.zip: comes with all binary files, samples, source code and documentation mentioned above.

All distribution files for this release can be found here.
