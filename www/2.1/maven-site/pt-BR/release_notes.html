2.1

Downloads

Todos os arquivos de distribuição para esta versão podem ser obtidos aqui.

Funcionalidades e mudanças

Algumas das novas funcionalidades e mudanças mais notáveis são:

  • POJOs podem ser usados para tratar todos os tipos de eventos
  • Os conversores do BeanUtils usam a mesma busca hierárquica dos outros componentes do genesis
  • A versão do Hibernate que acompanha o genesis agora é a 2.1.8
  • As referências diretas a classes do Commons JXPath foram deprecated. A próxima versão do genesis conterá suporte genérico a linguagens de script
  • Uma seção de FAQ foi adicionada à documentação

Os seguintes bugs foram corrigidos nessa versão:

  • Uma exceção é lançada quando um item numa list ligada a uma Collection ou array é selecionado (Issue # 192)
  • O reset do estado do form pode lançar uma NullPointerException sob condições específicas (Issue # 193)
  • Uma exceção é lançada se o objectField de um @DataProvider for uma array de qualquer subclasse de Object (Issue # 194)
  • O conversor padrão registrado para Object está errado (Issue # 202)

Uma lista completa das alterações ocorridas entre a 0.2 e a 2.1 pode ser obtida aqui.

A documentação completa para esta versão pode ser encontrada aqui.

0.2 (2.0)

Downloads

Todos os arquivos de distribuição para esta versão podem ser obtidos aqui.

Funcionalidades e mudanças

Diversas mudanças e funcionalidades novas fazem parte desta versão, entre outras podendo-se destacar:

  • Métodos remotos e transacionais são definidos através de anotações e não mais com marker interfaces + convenção de nomes para métodos
  • Implementação de @Criteria
  • Modo de execução local
  • Modelo de interface gráfica totalmente reformulado e muito mais expressivo e poderoso

Para obter a lista detalhada de mudanças nas versões intermediárias do 0.2.x (betas e RCs), clique aqui.
